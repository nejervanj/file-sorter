# file-sorter

A simple Python script that automatically sorts files into folders.

## Installation
1. Clone the repository:git clone https://github.com/nejervanj/file-sorter.git
2.  Open the folder:cd file-sorter
3.  Run the script:python Sorter.py
4.  ## Features
- Automatically sorts files into folders based on file type
- Easy to customize
- Works on Windows, macOS, and Linux

## Example
Before running:
- Downloads/
 - photo.jpg
 - song.mp3
 - document.pdf

After running:
- Downloads/
 - Images/photo.jpg
 - Music/song.mp3
 - Documents/document.pdf
